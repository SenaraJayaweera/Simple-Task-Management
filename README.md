# Task Ledger

A task manager built with React, Vite, and Tailwind CSS.

## Features
- Add tasks with title, description, priority, and due date
- Edit and delete tasks
- Update status (Pending, In Progress, Completed) inline
- Filter by priority, status, and free-text search
- Form validation on all required fields
- Responsive layout, persists to localStorage

## Getting started

```bash
npm install
npm run dev
```

Then open the printed local URL in your browser.

## Build for production

```bash
npm run build
npm run preview
```
